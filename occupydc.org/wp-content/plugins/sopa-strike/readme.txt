== Description ==

SOPA Strike will automatically redirect all users of your blog to the http://sopastrike.com homepage on Wednesday, January 18th. It automatically adds your blog name and URL to a list of protestors which will be featured on the website.

== Installation ==

Installing SOPA Strike is easy!

Automatic Installation:

1. Go to the Plugins tab in your Wordpress Admin.
1. Click *Add New*
1. Search for "SOPA Strike"
1. Click Install
1. All done!

Manual Installation:

1. Upload `sopastrike.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
