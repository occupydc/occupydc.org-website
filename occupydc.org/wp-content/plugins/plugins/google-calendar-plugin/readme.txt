== Description ==

This plugin show the google calendar in your weblog wordpress admin pannel.
You can choose the priveleges must an user needle for use Google Calendar Plugin and you can store a lot of calendar and see what you want in main pannel.
An administrator and olny he can share some calendar and all other users see that calendar in admin page.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Google Calendar Plugin` Directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use it!

WARNING! If you want upgrade, please see the upgrade section under Other Notes section.

== Upgrade ==

If you want upgrade from 1.2 to 1.3 version you must uninstall the 1.2 version and going to plugins section, upgrade with wordpress engine the plugin and return to Google Calendar section for use it. If you upgrade without uninstalling the previous version your DB contain one table named: google_calendar_privileges. You must remove that table in manual mode.
